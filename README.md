## Running the Project

Clone the repository:
git clone https://github.com/realario/ascent-blog

Install Node.js if you do not have it already.

Install dependencies:
cd ascent-blog/frontend
npm install
cd ../backend
npm install

Run the development servers in separate terminals:

Frontend:
cd ascent-blog/frontend
npm run dev

Backend:
cd ascent-blog/backend
npm run dev

The frontend should run on http://localhost:5173 and the backend should run on http://localhost:5000.
